idivisions
==========

Divide a contiguous integral sequence into shorter contiguous sequences.

Usage
-----

    idivisions(from=1, to, by=2L)
    
| Argument | Desciption |
| -------: | :--------- |
|   `from` | | passed to `seq` |
|     `to` | | passed to `seq` |
|     `by` | | passed to `seq` |
